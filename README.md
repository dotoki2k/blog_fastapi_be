## How to use:
Install Blog FastAPI with Python3.10 or higher.
```bash
   # using python3.10 to create environment.
   python3 -m venv venv
   
   # active environment.
   source venv/bin/activate
   
   # install requirement
   pip install -r requirement.txt
   
   # start project
   python3 main.py
```
